# Arduino Beginner Projects

This repository contains beginner-level Arduino projects and interfacing experiments created using Tinkercad simulations.

## Projects Included
- LED Blinking
- Ultrasonic Sensor
- LCD Interfacing
- Temperature Sensor

## Tools Used
- Tinkercad
- Arduino UNO
- Embedded Systems Concepts

## Components Used
- Sensors
- LEDs
- Jumper Wires
